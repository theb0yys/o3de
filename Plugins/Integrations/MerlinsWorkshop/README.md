# Optional Merlin's Workshop acquisition

`MerlinsWorkshop` is an optional editor-side integration for the official
Tainted Grail: The Fall of Avalon modding toolkit maintained by
`AR-Questline`.

The provider pins:

```text
release: 1.1.0
tag: v1.1.0
commit: 073bdab3e09d6adad5003339fc49b021738d71e6
Unity: 6000.0.60f1
Addressables: 2.7.4
```

The official repository uses Git LFS broadly and explicitly warns that
GitHub's autogenerated ZIP does not produce a usable toolkit. This package
therefore emits a deterministic **Git/LFS work order** rather than mirroring
the official repository or release assets.

## Commands

```powershell
python Plugins/Integrations/MerlinsWorkshop/Tools/merlins_workshop.py describe

python Plugins/Integrations/MerlinsWorkshop/Tools/merlins_workshop.py plan `
  --output C:\foa-build\merlin\work-order.json

python Plugins/Integrations/MerlinsWorkshop/Tools/merlins_workshop.py inspect-local `
  --checkout C:\Tools\merlin-workshop `
  --output C:\foa-build\merlin\inspection.json

python Plugins/Integrations/MerlinsWorkshop/Tools/merlins_workshop.py accept-license `
  --receipt C:\foa-build\merlin\inspection.json `
  --i-accept-the-official-license `
  --output C:\foa-build\merlin\qualified.json

python Plugins/Integrations/MerlinsWorkshop/Tools/merlins_workshop.py verify-receipt `
  --receipt C:\foa-build\merlin\qualified.json
```

## Acquisition boundary

The work order contains reviewed `git` and `git lfs` argument arrays with
`${MERLIN_DESTINATION}` as a non-private placeholder. Every step remains
`execution_allowed: false`. The current `ExternalToolchain` host does not
execute processes, and this package does not bypass that boundary.

A checkout qualifies only when:

- `.git/HEAD` is the exact pinned commit in detached form;
- all reviewed small anchor files match their exact Git blob SHA-1;
- both official licence PDFs are materialized from their exact Git LFS
  SHA-256 objects;
- the operator explicitly records acceptance of the official licence.

The receipt never stores the checkout path. Licence acceptance qualifies a
source observation only; it does not grant redistribution or runtime
authority.

## Permanent prohibitions

This package does not:

- clone, download, extract, launch Unity or bake a mod;
- accept GitHub's autogenerated ZIP;
- acquire an unpinned branch, tag or latest release;
- copy official source, first-party assets or licence PDFs into FOA-SDK;
- mutate catalog, source registry, game files or saves;
- promote evidence, deploy, sign, upload or publish;
- implement Mono or IL2CPP runtime behavior.

Candidate observations remain governed by
`GameInformationAcquisition` and the public `ExtensionAPI`.
