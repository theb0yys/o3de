# Foundation Frameworks

Tainted Framework contributes portable identity/configuration/diagnostic contracts, plugin-catalog fixtures,
exact Mono-versus-IL2CPP status, and a proven Mono compatibility row. Its BepInEx hosts remain runtime-only.

Avalon Core contributes a much larger canonical knowledge and planning surface. The first Avalon Core port must
inventory its engine registry and schemas, then select portable contracts individually. Runtime calls, game
assemblies, mutation adapters, and downstream consumer authority do not enter the Editor build graph.
